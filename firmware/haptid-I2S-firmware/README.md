# HapTID I2S demo

This program was made to work with the GUI in software\haptid-I2S-GUI.

## Hardware

We're using an ESP-WROOM-32 DevKit and a MAX98357A breakout board.

## Wiring

LRC -> any output pin
BCLK -> any output pin
DIN -> any output pin
GAIN -> +9dB if unconnected
SD -> GND
GND -> GND
VIN -> between 2.5V and 5.5V
